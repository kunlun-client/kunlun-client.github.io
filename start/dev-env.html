      # 开发环境

      # 1. Node.js（必需）

      终端执行 node -v 如果能打印出版本号说明已经安装，没有安装的到官网按说明安装。

      # 2. Yarn（推荐）

      用来替代 npm，介绍和安装方法见官网: https://yarnpkg.com

      # 3. VSCode（必需）

      • 安装 ESLint 插件，并设置 eslint.autoFixOnSave=true

      # 4. Chrome Extensions（推荐）

      • React Developer Tools
      • Redux DevTools Extension
