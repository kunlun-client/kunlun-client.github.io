      # 辅助工具

      提示

      本节介绍的工具非项目开发必须，可以按需自行选择。

      # 1. Shell

      • Oh My Zsh

      # 2. 终端

      • iTerm2 only for macOS
      • Cmder only for Windows

      # 3. 科学上网

      • shadowsocks
      • AgentNEO

      # 4. 网络抓包

      • Charles only for macOS
      • Wireshark

      # 5. 虚拟机

      • VirtualBox
        • Win8 专业版 镜像下载: cn_windows_8_1_x64_dvd_2707237.iso

      # 6. 设计

      • XMind 制作脑图
      • StarUML UML 建模
      • Mockplus 原型设计
      • PxCook 自动标注

      # 7. 图像处理

      • base64-image.de 在线图片转成 base64

      # 8. 工具社区

      • 利器 利器采访优秀的创造者，邀请他们来分享工作时所使用的工具，以及使用工具的方式和原则。每周一与周四更新。
